# MessageBox_splash  
MessageBox splash like computer virus.  
  
## Usage  
Execute main.bat  
  
## ※This is for Windows

## Custom  
If you want to change number of times to loop , change this (this is 40 loop -> 100 loop) ↓  
### in main.bat
```bat
for /l %%i in (1,1,40) do start alert.vbs
```

```bat
for /l %%i in (1,1,100) do start alert.vbs
```
